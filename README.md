# Thumb Timer

A minimal, privacy-friendly timer app designed for one-handed use.

## Features

- Preset timers
- Custom timer
- Background timer support
- Foreground notification
- Pause and reset from notification
- Vibration when the timer finishes
- Material 3 UI
- Dark/light theme support
- No internet permission
- No tracking
